# Upsampling e Downsampling

Esta é uma atividade para a matéria "Tópicos Especiais em Engenharia de Computação 3", cursada por mim no SLS de 2020 da UFBA.

No Jupyter Notebook aqui desenvolvido, foi programado um algoritmo para efetuar upsampling de fator 2 ou downsampling de fator 2 para sinais de áudio `.wav`.

A função desenvolvida no código retorna:
*   Player de áudio para o arquivo original;
*   Player de áudio para o arquivo resampleado;
*   Gráfico do sinal de áudio original no domínio do tempo;
*   Gráfico do sinal de áudio resampleado no domínio do tempo;
*   Espectrograma do sinal de áudio original;
*   Espectrograma do sinal de áudio resampleado.

O objetivo da atividade é promover a reflexão em relação ao teorema da amostragem de Nyquist e observar na prática os efeitos que a variação da taxa de amostragem de um sinal pode provocar.
